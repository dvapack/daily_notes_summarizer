# Простой текст в Markdown

Это пример обычного абзаца. Здесь можно писать любой текст.

## Списки

Маркированный список:
- Первый пункт
- Второй пункт
- Третий пункт

Нумерованный список:
1. Первый пункт
2. Второй пункт
3. Третий пункт
